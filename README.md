# ES-A-StarWars-Theme
It's a StarWars inspired theme, made for fans of the series, for better experience, turn-on navigation sounds!

# Aspect Ratio :
- Made for 4/3 screens, 640x480 resolution,  Handheld Console
  
# Gamelist View Style :
- Basic
- Detailed
- Grid
